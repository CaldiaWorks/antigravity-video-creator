# Remotion x AI 動画生成プロジェクト

このプロジェクトは、**React**ベースの動画生成フレームワークである [Remotion](https://www.remotion.dev/) と、**AIエージェント（Antigravity）**を組み合わせ、対話を通じて自律的に動画を作成・編集することを目的とした実験プロジェクトです。

## 🚀 プロジェクトの目的
- AIチャットを通じた直感的な動画作成・構成の自動生成
- Remotion Skillを活用したベストプラクティスに基づく高度なビデオエフェクトの実装
- 自動レンダリングパイプラインの構築

## 🛠 使用技術
- **Remotion**: Reactによる動画プログラム
- **Antigravity (AI)**: コード生成・編集および自律的なタスク遂行
- **Remotion Skill**: AIがRemotionを最適に使いこなすためのルールセット

---

## 💡 Remotion Skill について
本プロジェクトには `remotion-best-practices` というSkillが導入されています。これにより、AIは以下のような高度な実装を自律的に行えます。

### Skillでできること（日本語解説）
| カテゴリ | 解説 |
| :--- | :--- |
| **アニメーション** | Spring（バネ）アニメーションやイージングを用いた滑らかな動きの実装 |
| **シーケンス制御** | 動画のタイミング、遅延、クリッピングの正確な調整 |
| **アセット管理** | 画像、動画、音声、Lottie（アニメーション）の最適な読み込み |
| **タイポグラフィ** | Google Fontsの利用や、テキストの折り返し・サイズ自動調整 |
| **データ可視化** | チャートやグラフを動的に生成し動画に組み込む |
| **キャプション生成** | 音声からの字幕生成や、TikTokスタイルのハイライト表示 |
| **3D / 地図** | React Three Fiberを用いた3D表現や、Mapboxによる動的な地図アニメーション |

## ⚡️ クイックスタート

### プレビューの起動
動画をブラウザでリアルタイムに確認します。
```bash
npm start
```

### ビデオの書き出し (MP4)
現在のコードを元に動画をレンダリングします（`ffmpeg`が必要）。
```bash
npm run build
```

---

## 🎨 AIへの指示の例
Antigravity（AI）に対して、以下のような指示を送ることで動画をカスタマイズできます。

- 「背景をもっと近未来的な青いグラデーションにして、テキストを中央で弾ませて」
- 「画面の下半分にスクロールする字幕を追加して」
- 「30秒の動画にして、10秒地点で画像をフェードインさせて」
